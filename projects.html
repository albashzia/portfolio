ATM Simulator (Python)

A console-based ATM simulator developed in Python that replicates core banking operations through a secure, menu-driven interface. Users can authenticate using an account number and PIN, view balances, deposit and withdraw funds, pay utility bills, change their PIN, and receive a detailed transaction receipt at the end of each session.

The project demonstrates Python fundamentals including functions, loops, conditionals, input validation, exception handling, and in-memory data management using lists and dictionaries. It also implements secure login logic, session tracking, transaction history, and realistic banking workflows without relying on external libraries or databases.

Caesar Cipher (Python)

A console-based Caesar Cipher application developed in Python that allows users to encrypt and decrypt text using the classical substitution cipher. The program features a menu-driven interface where users can choose between encryption, decryption, or exiting the application while preserving letter case and leaving non-alphabetic characters unchanged.

The project demonstrates procedural programming, modular function design, string manipulation, ASCII character processing using ord() and chr(), conditional logic, wrap-around arithmetic with the modulus operator, and user input handling. The encryption and decryption logic is separated into dedicated modules to promote code organization and maintainability.
